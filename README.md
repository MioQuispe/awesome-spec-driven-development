# Awesome Spec Driven Development [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A Selective list of awesome resources for Spec Driven Development

## Tools

- [Kiro](https://kiro.dev/) - AI-powered IDE with built-in spec-driven development capabilities.
- [OpenSpec](https://openspec.dev/) - OpenSpec aligns humans and AI coding assistants with spec-driven development so you agree on what to build before any code is written. 
- [SpecKit](https://github.com/github/spec-kit) - An open source toolkit that allows you to focus on product scenarios and predictable outcomes instead of vibe coding every piece from scratch.
- [Tessl](https://tessl.io/) - Spec-as-source development platform that generates code directly from specifications. 
- [AINativeDev](https://ainativedev.io/podcast) - Podcast, news & community focused on AI Native & Spec driven development
